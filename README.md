# Bitcoin Toolkit

A Next.js app with tools to help grow your Bitcoin stack and outpace inflation.

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open http://localhost:3000 in your browser.

## Project Structure

- **app**  
  Routes, pages, layouts, and API endpoints

- **components**  
  Reusable UI pieces like cards, navbars, and forms

- **services**  
  External API interaction such as Bitcoin price or wallet data fetching

- **lib**  
  Helpers, utilities, config, constants, and formatting logic

- **public**  
  Static assets like images and icons
